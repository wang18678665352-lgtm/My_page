# 扫雷Pro - 经典扫雷游戏与AI助手

这是一个基于网页的扫雷游戏，包含了与微软经典扫雷相似的游戏体验，同时添加了智能AI助手功能，可以模拟人类思路进行扫雷。

## 游戏特性

- **经典扫雷体验**：提供初级(9×9)、中级(16×16)和高级(30×16)三种难度
- **智能AI助手**：
  - 自动扫雷功能：AI会模拟人类逻辑思维自动完成游戏
  - 单步AI模式：每一步都可以查看AI的决策过程
  - 遵循人类思维模式：不会使用作弊算法，确保所有地雷都能通过逻辑判断确定
  - 可视化思考过程：每步之间有适当延迟，让人类玩家能看清AI的扫雷过程
- **现代化界面**：使用Tailwind CSS构建的响应式界面
- **游戏统计**：显示剩余地雷数量和游戏时间
- **键盘快捷键**：支持键盘操作，提升游戏体验

## 技术栈

- HTML5
- JavaScript
- Tailwind CSS (通过CDN引入)
- Font Awesome (通过CDN引入)

## 如何运行游戏

1. 直接双击`index.html`文件在浏览器中打开游戏
2. 或者使用任何Web服务器托管此文件夹

## 游戏操作说明

- **左键点击**：翻开格子
- **右键点击**：标记/取消标记可能的地雷
- **难度选择**：可以在游戏开始前或重新开始时选择不同难度
- **重新开始**：点击中间的刷新按钮或按R键
- **AI扫雷**：点击"AI扫雷"按钮或按A键，AI将自动进行扫雷
- **单步AI**：点击"单步AI"按钮或按S键，AI将执行一步扫雷操作

## AI工作原理

AI模拟人类思路进行扫雷，主要采用以下策略：

1. **确定性逻辑**：寻找100%安全的格子和100%确定的地雷
2. **概率分析**：对不确定的格子进行概率计算，选择最安全的格子
3. **规则遵循**：不在一个九宫格内任何一个格子都没被扫开的情况下翻开九宫格的中心格
4. **延迟执行**：每步操作之间有适当延迟，让人类玩家能够观察和理解AI的决策过程

## 开发说明

- 游戏逻辑与UI分离，便于维护和扩展
- AI算法设计尽量模拟人类思维，避免使用直接读取地雷位置的作弊方式
- 代码中添加了详细注释，便于理解和二次开发

## 注意事项

- 为了获得最佳体验，请使用现代浏览器(Chrome、Firefox、Edge等)
- 游戏数据仅保存在浏览器内存中，刷新页面将重置游戏状态
- AI扫雷功能设计为模拟人类思维，可能不会在所有情况下都能成功完成游戏
